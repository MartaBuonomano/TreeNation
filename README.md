# Getting Started

## Dependencies
- node
- npm
- Cucumber (Gherkin) Full Support

## Installing
```shell script
$ npm init
$ npm install
$ npm install cypress --save-dev
$ npm i cypress-real-events
```

## Run Tests Headless
```shell script
$ npm run test
```

## Open Cypress console
```shell script
$ npx cypress open
```
